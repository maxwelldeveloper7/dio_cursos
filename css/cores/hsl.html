Cores

Existem várias formas de definirmos cores utlizando o CSS. Elas são:

  • Cores pré-definidas
  • Palavra-chaves "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

HSL (hue, saturation, ligthness)
  • Hue (matriz): é o grau da roda de cores de 0 a 360 graus!
    • 0 ou 360 vermelho
    • 120 verde
    • 240 azul
  • Saturation (saturação): valor percentual.
    • 0% um tom de cinza
    • 100% cor total
  • Ligthness (luminosidade): luminosidade da cor.
    • 0% preto
    • 100% branco
